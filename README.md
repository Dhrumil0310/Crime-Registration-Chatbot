# Crime Registration and Awareness Chatbot
Vigilante Bot tryna help eradicate crime from this world

### Techs Used
* React.js, Redux
* Firebase - Auth & Cloud Firestore
* Bootstrap
* DialogFlow

### Setup
To run this project clone it and then :
```
$ npm install
$ npm start
```
